# 오류 메세지

```
/usr/lib/ruby/2.7.0/socket.rb:201:in 'bind': Address already in use - bind(2) for 127.0.0.1:4000 (Errno::EADDRINUSE)
```

---

# 원인

이미 포트 4000을 사용하는 프로세스가 존재할 때 발생하는 에러로 파악하고 있다.

---

# 해결 방법

해당 포트를 사용하는 프로세스를 종료하면 된다.

```
sudo lsof -wni tcp:4000
```

```
sudo kill -60048 PID
```
