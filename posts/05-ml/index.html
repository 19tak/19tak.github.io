게시 2021-09-15 업데이트 2022-10-16
By 19tak

본 포스팅은 “파이썬 머신러닝 완벽가이드” - (위키북스, 권철민 지음)을 바탕으로 공부한 내용을 정리한 것입니다.

  1. 단일 값 추출
  2. 슬라이싱
  3. 팬시 인덱싱
  4. 불린 인덱싱

1. 단일 값 추출

원하는 위치의 인덱스 값을 지정하면 해당 위치의 데이터가 반환.

ndarray[n]이면 n+1번째 array의 index 위치의 데이터값을

2차원에서는 ndarray[n,m]이면 n+1번째 row, m+1번째 column 이런식으루

2. 슬라이싱

ndarray[0:n]이면 0번째 인덱스부터 n-1번째 인덱스까지 추출

2차원에서는 단일값추출과 유사하게 row, col 각각의 인덱스에 슬라이싱 적용

ndarray[1:n, :] 이런식으루

3. 팬시 인덱싱 (Fancy Indexing)
